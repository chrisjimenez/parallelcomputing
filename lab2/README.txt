README FOR PARALLEL COMPUTING LAB 2

To compile the program, please type the following at the command line:

—>gcc -g -Wall -fopenmp -o tsm tsm.c

To run the program, you must type in the file name and the number of cities in that file. For example, for the cities3.txt, you would type the following at the command line:

—> ./tsm cities3.txt 3
